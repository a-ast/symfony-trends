Core team

Commits of all contributors mentioned on Symfony Core Team page are marked on charts as 'core team commits'.

Maintenance commits

Symfony core team does a hard job maintaining git repositories. Significant part of all commits is merging pull requests, bumping versions, fixing and reverting stuff, etc. Besides this, maintenance tools generates automatic commits. In order to show real contributions, all maintenance commits have been filtered using these regular expressions:
  • ^minor #
  • ^bug #
  • ^feature #
  • ^merged branch
  • ^Merge branch
  • ^merged
  • ^Merge remote-tracking branch
  • ^Merge pull request #
  • ^updated VERSION
  • ^update CONTRIBUTORS
  • ^updated CHANGELOG
  • ^bumped Symfony version to
  • ^updated version to
  • ^fix merge
  • ^fixed merge
  • ^fixed bad merge
  • ^Revert "
  • ^Revert #
